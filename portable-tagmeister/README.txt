# Tagmeister Portable

This is a portable version of the Tagmeister application. To run the application, simply double-click on the `tagmeister.exe` file.

## Files

- `tagmeister.exe`: The main executable file
- `tagmeister_lib.dll`: The main library file
- `resources\icon.ico`: The application icon
- `dist\`: The web assets for the application

## Notes

- This portable version contains all the necessary files to run the application without installation.
- You can move this entire folder to any location on your computer or to a USB drive.
- All data will be stored in the same location as the application.
